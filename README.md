## latex file modifier and pdf creator 

## Installation

### Method 1: Using install script (Linux/macOS)
```bash
chmod +x install.sh
./install.sh
```

### Method 2: Manual installation
1. Install Python 3.7+
2. Install LaTeX (TeX Live or MiKTeX)
3. Run:
```bash
pip install -e .
```

## Usage
```bash
latex-selector
```
